# IRChat
Current:
Primitive TCP IRC style chat written in node

## Usage:
Server:
```javascript
node server.js
```

Client:
```javascript
node cli.js <username>@<hostname> [options]
```

Ambitions:
- Highly customizeable clients
- Highly customizeable servers
- Secure connections

Featurs:
- Web-Sockets
- Easy customizeable clients
- AES-192-CBC Encryption
- FTP Server
